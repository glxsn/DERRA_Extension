# DERRA_Extension
GTA 5 Mod: DERRA's Expansion


How to use this mod?
1.Install the original DERRA and its dependencies
2.Place the DERRA_Extension.dll file into the "scripts" folder
3.Launch the game

This extension is developed based on DERRA version 1.2.5. If you want to use this source code, you will need the following dependencies.
1.The original DERRA
2.ScriptHookVDotNet3 version 3.6.0.135 (consistent with the original DERRA)
3.LemonUI.SHVDN3 version 2.2.0.0
